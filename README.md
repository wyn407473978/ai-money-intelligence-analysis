# AI赚钱情报分析 Skill

分析 AI 新闻/热点采集程序输出的 JSON 数据，从中筛选出适合「AI赚钱情报站」使用的内容选题、商业机会、短视频爆款方向和风险提醒。

## 用途

当 `ai_news_fetcher.py` 采集脚本运行完毕后，对输出的 JSON 数据进行二次分析，产出面向普通人的赚钱情报报告。

## 适用场景

- `ai_news_fetcher.py` 运行完毕，JSON 数据已获取
- 需要为「AI赚钱情报站」生成日报/周报
- 用户说"帮我分析今天的 AI 热点"或类似指令

## 评分体系

| 等级 | 含义 |
|------|------|
| **S** | 强证据、多源报道、传播广、有明确普通人参与路径 |
| **A** | 满足多项条件，有较明确参与路径 |
| **B** | 技术趋势或行业背景，不适合直接起号 |
| **C** | 噪音内容，过滤排除 |

## 机会类型

- **流量型** - 适合涨粉、蹭热点、做短视频/图文
- **产品型** - 适合做资料包、模板包、工具清单、课程
- **服务型** - 适合接单、咨询、企业交付
- **观察型** - 适合做周报、深度分析、趋势储备

## 核心规则

- 单一来源热点最高只能评 A
- 技术内容（benchmark/论文）最高只能评 B
- 医疗/金融/法律高风险类别必须加风险提醒
- 爆款钩子不得含未经证实的硬事实
- `publish_now=wait` 的内容不能强推变现

## 输出报告结构

1. 今日结论（3-5条 bullet）
2. S级机会（最多2条）
3. A级机会（最多3条）
4. B级观察（最多3条）
5. 今日不适合做的内容
6. 今日最适合发布的3个选题
7. 今日选题组合策略
8. 发布前人工复核清单

## 使用方式

在 Hermes Agent 中加载此 Skill：

```
/skill ai-money-intelligence-analysis
```

然后提供 `ai_news_fetcher.py` 输出的 JSON 数据，即可获得完整的赚钱情报报告。
